# LLM Ontology Reuse Evaluation

## Overview
This repository contains the experimental framework, datasets, prompts, and analysis used to evaluate **ontology reuse in ontologies generated by Large Language Models (LLMs)**.

Ontology reuse is a fundamental principle in ontology engineering, promoting interoperability, knowledge sharing, and reducing redundant modeling efforts. However, in practice, reuse is difficult to achieve. While Large Language Models are increasingly used for automated ontology generation, their ability to **explicitly reuse existing ontologies** remains largely unexplored.

This project introduces an **evaluation framework for measuring ontology reuse** in LLM-generated ontologies and analyzes how different prompting strategies influence reuse behavior.

---

## Research Objective
The goal of this work is to investigate whether LLMs reuse existing ontologies when generating new ontological models and to measure the degree of reuse using multiple evaluation dimensions.

The framework evaluates reuse across the following dimensions:

- **Lexical Reuse** – reuse of terminology and class labels from existing ontologies  
- **Structural Reuse** – reuse of hierarchical structures and class relationships  
- **Semantic Reuse** – reuse of subclass relationships that maintain the conceptual meaning of classes
- **Logical Consistency** – preservation of logical correctness and reasoning compatibility  
- **Reuse Depth** – extent of axiomatic and structural reuse

---

## Evaluated Models
The experiments analyze ontology generation using multiple LLMs:

- GPT-4.1  
- GPT-4o-mini  
- Qwen2.5-7B  
- Llama-3.1-8B  

---

## Reference Ontologies
The experiments use well-established ontologies from the **IoT and energy domains**, including:

- **SAREF (Smart Applications REFerence Ontology)**
- **Fiesta-IoT Ontology**

These ontologies serve as **reference knowledge bases** to evaluate reuse in generated ontologies.
